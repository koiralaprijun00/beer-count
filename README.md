A playful, mobile-first beer tracking starter built with Next.js (App Router), Firebase Auth, and Cloud Firestore. Add your beers, log each time you drink them, and keep tabs on your stats.

## Getting started

1. Install dependencies:

```bash
npm install
```

2. Add your Firebase config to `src/lib/firebase.ts` by replacing the placeholder values.

3. Run the dev server:

```bash
npm run dev
```

Visit http://localhost:3000 to sign up, log in, search beers, and track counts.

## Tech stack
- Next.js 14 (App Router) + TypeScript
- Firebase Auth (email/password)
- Cloud Firestore for beer stats
- Tailwind CSS for a funky, friendly UI
